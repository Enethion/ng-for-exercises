Ćwiczenia z *ngFor

1. Wypisz tablicę 'frameworks' za pomocą *ngFor i dodaj index:

  • {{i}}. {{ framework }}

2. Wypisz model i markę samochodu z tablicy 'cars' za pomocą *ngFor:

  • {{ car.mark }} {{ car.model }}

3. Wypisz model i markę samochodu z obiektu 'carsObj' za pomocą *ngFor i pipe keyvalue:

  • {{ car.key }} {{ car.value }}

4. Wypisz z tablicy 'frameworks' tylko pierwszy i ostatni element. Dodatkowo należy użyć *ngIf:

  • {{ framework }}
    •

5. Wypisz z tablicy 'frameworks' elementy, które mają parzysty indeks. Dodatkowo należy użyć *ngIf (jeżeli już wiesz jak używać):

  • {{ framework }}
    •

6. Wypisz z w pętli obiektu 'persons' dla każdego użytkownika: Id, name, city, company name:

  • {{ user?.id }}. {{ user?.name }} {{ user?.address?.city }} {{ user?.company?.name }}
